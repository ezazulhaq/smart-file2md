### Core Functionality
- **Smart PDF Conversion**: Automatically converts text-based PDFs to clean Markdown format
- **OCR Fallback**: Seamlessly falls back to OCR (Tesseract) for scanned documents and image-based PDFs
- **Batch Processing**: Convert multiple PDF files at once with progress tracking
- **Recursive Directory Processing**: Process entire directory trees with a single command
- **Flexible Output**: Specify custom output directories or use automatic organization

### Architecture Highlights
- **Object-Oriented Design**: Built with SOLID principles and design patterns
  - Strategy Pattern for different conversion methods
  - Factory Pattern for automatic converter selection
  - Template Method for consistent conversion workflow
- **Type-Safe**: Full type hints throughout the codebase
- **Extensible**: Easy to add new converter types or extend functionality
- **Well-Tested**: Comprehensive unit and integration tests

### Command-Line Interface
```bash
# Simple conversion
smart-file2md document.pdf

# Batch conversion with options
smart-file2md *.pdf -o output/ --max-pages 10

# Force OCR for all files
smart-file2md scanned.pdf --force-ocr

# Recursive directory processing
smart-file2md /path/to/pdfs/ --recursive
```

### Python API
```python
# Simple usage
from markdown_convert import convert_pdf
convert_pdf('document.pdf')

# Advanced usage with configuration
from markdown_convert import ConverterConfig, ConverterFactory
from pathlib import Path

config = ConverterConfig(
    output_dir=Path('output/'),
    max_pages=20,
    force_ocr=False
)

converter = ConverterFactory.create(Path('document.pdf'), config)
output_path = converter.convert(Path('document.pdf'))
```

## ğŸ”§ Configuration Options

| Option | Description | Default |
|--------|-------------|---------|
| `--output-dir` | Output directory for markdown files | `./markdown/` |
| `--max-pages` | Maximum pages to process per PDF | All pages |
| `--force-ocr` | Force OCR even for text-based PDFs | `False` |
| `--recursive` | Process directories recursively | `False` |
| `--overwrite` | Overwrite existing output files | `False` |
